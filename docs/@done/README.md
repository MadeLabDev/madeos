# MADE (OS) Documentation

This directory contains documentation for the MADE (OS) project.

## Structure

- **rag/**: Documentation related to Retrieval-Augmented Generation (RAG) features, including setup, activation, architecture, and guides.
- **setup/**: Setup and deployment guides, including Cloudflare Pages setup and file manifests.
- **architecture/**: Architectural documentation, including vector storage strategies and marketing implementation plans.
- **done/**: Completed or archived documentation.
